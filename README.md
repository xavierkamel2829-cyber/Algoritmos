Este es un programa en Python para calcular recomendaciones nutricionales y de suplementación (calorías diarias, proteína y creatina) para personas con objetivos atléticos. El script solicita datos por consola, valida entradas y devuelve un "plan de suplementación" básico por usuario.

Características 

Cálculo de la Tasa Metabólica Basal (Harris-Benedict) con factor de actividad fijo.
Ajuste de calorías según objetivo: Volumen, Mantenimiento o Definición.
Recomendación de proteína (g/kg) y creatina (g/kg) basada en el peso.
Validación de entradas (sexo, números) y soporte para evaluar múltiples usuarios en una sola ejecución.

Uso
descarga Python 3.x
Clona o descarga el repositorio.
Ejecuta el script desde la terminal:
python3 "Proyecto Informatica.py"
